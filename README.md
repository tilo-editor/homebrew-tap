# Homebrew Tap for Tilo

Install Tilo with Homebrew:

```sh
brew tap tilo-editor/tap
brew install tilo
```

Tilo is the tiled terminal editor. The public distribution repo is:

```text
https://github.com/tilo-editor/tilo
```

## Maintainers

The formula installs prebuilt release archives from GitHub Releases. The private
source repo release workflow updates `Formula/tilo.rb` when a new Tilo release
is published.

The formula supports:

- macOS arm64
- macOS x86_64
- Linux arm64
- Linux x86_64

# Homebrew Tap for Tilo

Install Tilo with Homebrew:

```sh
brew tap tilo-editor/tap
brew install tilo
```

Tilo is the tiled terminal editor. The public distribution repo is:

```text
https://github.com/tilo-editor/tilo
```

## Maintainers

The formula installs prebuilt release archives from GitHub Releases. Update
`Formula/tilo.rb` whenever a new Tilo release is published.

For `v0.1.0`, the formula supports:

- macOS arm64
- macOS x86_64
- Linux arm64
- Linux x86_64
